Keyboard Shortcuts for Mac

Library and navigation
Start or stop editing Enter
Delete selection Delete
Quit Membrane Command + Q
Close the Membrane window Command + W
Open the Info window for selection Command + I
Add projects to your Membrane library Command + O
Select all the projects in your Membrane library Command + A
Deselect a selection Command + Shift + A, Command + D, ESC
Hide the Membrane window Command + H
Hide all other applications Option + Command + H
Editor
Move canvas Space + Drag
Zoom in Command + "+"
Zoom out Command + "-"
Actual size Command + 0
Edit the next object Shift + Tab
Edit the next object Tab
Select all the objects Command + A
Deselect a selection Command + Shift + A, Command + D, ESC
Bring forward selection Command + ]
Bring to front selection Command + Shift + ]
Send backward selection Command + [
Send to back selection Command + Shift + [
Move selection 1 pixel Up, Down, Left, Right Arrow
Move selection 10 pixel Shift + Up, Down, Left, Right Arrow
Change selection size 1 pixel Command + Up, Down, Left, Right Arrow
Change selection size 10 pixel Command + Shift + Up, Down, Left, Right Arrow
Go to the next or previous project Command + Up Arrow or Down Arrow
Cut selection Command + X
Copy selection Command + C
Paste selection Command + V
Delete selection Delete
Undo your last change Command + Z
Redo your last change Command + Shift + Z
Save Membrane data Command + S
