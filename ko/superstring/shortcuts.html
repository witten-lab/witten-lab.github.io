재생 시작 또는 일시 중지 Space bar
편집 시작 또는 중지 Enter
소리 음소거 (노래 계속 재생) M
선택 항목 삭제 Delete
재생 중지 Command + .

다음 또는 이전 노래로 이동 Command + 위쪽 또는 아래쪽 화살표
Superstring 종료 Command + Q
Superstring 창 닫기 Command + W
선택 항목 정보 창 열기 Command + I
Superstring 보관함에 노래 추가 Command + O
Superstring 보관함의 모든 노래 선택 Command + A
선택 항목 선택 취소 Command + Shift + A
Superstring 윈도우 숨기기 Command + H
다른 모든 응용 프로그램 숨기기 Option + Command + H
에디터

다음 클립 편집 Tab
모든 클립 선택 Command + A
선택 항목 선택 취소 Command + Shift + A

선택 항목 위 또는 아래로 1초 이동 Shift + 위쪽 또는 아래쪽 화살표
선택 항목 시작 시간 변경 0.1초 Option + 위쪽 또는 아래쪽 화살표
선택 항목 시작 시간 변경 1초 Option + Shift + 위쪽 또는 아래쪽 화살표
선택 항목 종료 시간 변경 0.1초 Command + 위쪽 또는 아래쪽 화살표
선택 항목 시작 시간 변경 1초 Command + Shift + 위쪽 또는 아래쪽 화살표
위에 클립 삽입 Command + Shift + I
아래에 클립 삽입 Command + I
분배 Command + D
선택 항목 그룹화 Command + G
선택 항목 그룹 해제 Command + Shift + G
선택 항목 잘라내기 Command + X
선택 항목 복사 Command + C
선택 항목 붙여 넣기 Command + V
선택 항목 삭제 Delete
마지막 변경 사항 실행 취소 Command + Z
마지막 변경 사항 다시 실행 Command + Shift + Z
Superstring 데이터 저장 Command + S
