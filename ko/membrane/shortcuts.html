Mac 키보드 단축키

라이브러리 및 탐색
편집 시작 또는 중지 Enter
선택 항목 삭제 Delete
Membrane 종료 Command + Q
Membrane 창 닫기 Command + W
선택 항목 정보 창 열기 Command + I
Membrane 보관함에 새로운 프로젝트 추가 Command + O
Membrane 보관함의 모든 프로젝트를 선택 Command + A
선택 항목 선택 취소 Command + Shift + A, Command + D, ESC
Membrane 창 숨기기 Command + H
다른 모든 응용 프로그램 숨기기 Option + Command + H
에디터
캔버스 이동 Space + Drag
확대 Command + "+"
축소 Command + "-"
실제 크기 Command + 0
이전 객체 편집 Shift + Tab
다음 객체 편집 Tab
모든 객체 선택 Command + A
선택 항목 선택 취소 Command + Shift + A, Command + D, ESC
선택 항목 앞으로 가져오기 Command + ]
선택 항목 가장 앞으로 가져오기 Command + Shift + ]
선택 항목 뒤로 보내기 Command + [
선택 항목 가장 뒤로 보내기 Command + Shift + [
선택 항목 이동 1픽셀 위쪽, 아래쪽. 왼쪽, 오른쪽 화살표
선택 항목 이동 10픽셀 Shift + 위쪽, 아래쪽. 왼쪽, 오른쪽 화살표
선택 항목 크기 변경 1픽셀 Command + 위쪽, 아래쪽. 왼쪽, 오른쪽 화살표
선택 항목 크기 변경 10픽셀 Command + Shift + 위쪽, 아래쪽. 왼쪽, 오른쪽 화살표
다음 또는 이전 프로젝트로 이동 Command + 위쪽 또는 아래쪽 화살표
선택 항목 잘라내기 Command + X
선택 항목 복사 Command + C
선택 항목 붙여 넣기 Command + V
선택 항목 삭제 Delete
마지막 변경 사항 실행 취소 Command + Z
마지막 변경 사항 다시 실행 Command + Shift + Z
Membrane 데이터 저장 Command + S
