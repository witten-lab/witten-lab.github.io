Superstring installation problem

Occasionally, Windows users may run into an error where the Superstring installer fails. To resolve this issue:

  1. Uninstall Superstring and Adobe Air.
  2. Download and install the latest version of Adobe Air.
  3. Download and install the latest version of Superstring

Can I use my own fonts?

Unfortunately, Superstring does not support to use custom fonts in your computer. However, we continue to add fonts to Superstring every update. We strive to diversify our selection to give our users as many choices as possible. If you have fonts you want us to add, simply contact us and we will do our best to add it to Superstring.

 

Error: "The publisher of Superstring couldn't be verified."

While downloading Superstring, if you are getting an error "The publisher of Superstring couldn't be verified." or "The publisher of this program couldn't be verified", please try to download it again using a different browser like Chrome or Firefox.

 

What happens if I don't renew my license?

If you don't want to renew your license after it expires, you can continue to use the version of Superstring that you have, for as long as you want, but you won't be able to receive any further updates. If you decide you want to renew at a later date, you're welcome back at any time.

 

When will I have to renew my license?

Your license is valid for 1 year after the date of purchase so if you bought your license today {{site.time | date: "%b %-d, %Y"}}, you'll get all the regular updates that are released up to {{site.time | date: "%b %-d"}}, {{site.time | date: "%Y" | plus:1}}. To receive updates after that date, you'll need to renew your license.

 

Can I install Superstring on multiple computers?

When you purchase Superstring, your license entitles you to use Superstring on one computer only.

 

Superstring crashes when a lyric video is exporting

It might be memory or performance-related. Please try these suggestions. The most common causes for that problem can be solved with these few steps:

  • Reduce the number of text styles. Working with many text styles use more system resources.
  • Reduce the number of background images. Working with many background images use more system resources.
  • Remove effects. Effects uses a lot of system resources.
  • Export a lyric video in HD 720p. HD 720p is sufficient in most cases such as YouTube and Facebook.

If the problem persists after you've tried these, please contact us.

 

How do I play my lyric video(FLV)?

Check out VLC media player. It's a free and open source cross-platform multimedia player that plays most multimedia files.

 

My lyric video .flv is so huge

.flv format is an option if .mp4 format is not working with your computer. It's uncompressed video, so that's why it's so huge. If you need a smaller file size you can just convert the video to other format with these free applications.

  • Mac: Smart Converter
  • Windows: Freemake Video Converter
