Mac

Playback and navigation
Start or pause playing Space bar
Start or stop editing Enter
Mute the sound (song keeps playing) M
Delete selection Delete
Stop playing selection Command + .
Go to the next or previous clip Right or Left Arrow, Up or Down Arrow
Increase or decrease the volume Command + Right or Left Arrow
Go to the next or previous song Command + Up Arrow or Down Arrow
Quit Superstring Command + Q
Close the Superstring window Command + W
Open the Info window for selection Command + I
Add songs to your Superstring library Command + O
Select all the songs in your Superstring library Command + A
Deselect a selection Command + Shift + A
Hide the Superstring window Command + H
Hide all other applications Option + Command + H
Editor
Zoom in Command + "+"
Zoom out Command + "-"
Actual size Command + 0
Edit the previous clip Shift + Tab
Edit the next clip Tab
Select all the clips Command + A
Deselect a selection Command + Shift + A
Move up or down selection 0.1 second Up or Down Arrow
Move up or down selection 1 second Shift + Up or Down Arrow
Change selection start time 0.1 second Option + Up or Down Arrow
Change selection start time 1 second Option + Shift + Up or Down Arrow
Change selection end time 0.1 second Command + Up or Down Arrow
Change selection end time 1 second Command + Shift + Up or Down Arrow
Insert clips above Command + Shift + I
Insert clips below Command + I
Distribute Command + D
Group selection Command + G
Ungroup selection Command + Shift + G
Cut selection Command + X
Copy selection Command + C
Paste selection Command + V
Delete selection Delete
Undo your last change Command + Z
Redo your last change Command + Shift + Z
Save Superstring data Command + S
